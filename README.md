# ✈️ Layover Friends

An AI-powered airport layover networking app that helps travelers connect during layovers around the world.

## 🚀 Features
- Search global airports
- Find travelers with matching layover times
- AI-powered smart matching
- Send connection requests

## 🛠️ Tech Stack
- **Frontend:** Flutter (Android + Web)
- **Backend:** Supabase (PostgreSQL + Auth)
- **AI Service:** FastAPI + OpenAI Embeddings

## 📱 Screens
Home → Airport Search → Matches → Connect → Success
